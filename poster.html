API Workflow
Request sequence
Every search runs this sequence in order. APIs are stateless and non-overlapping.
01Exa Research API
strategy

Generates target repos, poach companies, and EEA signal definitions from role spec. Returns structured JSON with citations. No hallucinated repos.

02Parallel Task API
parallel

Researches all target companies simultaneously. Returns headcount, tech stack, hiring signals, attrition patterns per company.

03GitHub REST API
proof

Mines contributors from each target repo. Pulls commit counts, maintainer status, contribution recency.

04Exa Search + Answer
discovery

Neural search matches candidates by embeddings. Answer API synthesizes "what is this person known for?" with citations per candidate.

05Harmonic + Scoring
output

Adds market and ecosystem context. Scores 0–100 against EEA criteria. Flags hidden gems. Pipeline complete.
